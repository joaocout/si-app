Para executar o código fonte, é necessário:
    NodeJS (desenvolvido na versão 14.17.0)
    NPM (desenvolvido na versão 6.14.13)

===========================================================

Com esses requisitos, basta instalar o expo-cli:
    npm install -g expo-cli

E o Yarn:
    npm install -g yarn

E executar o Yarn para baixar as dependencias (pode demorar um pouco) e o servidor do expo na pasta:
    yarn 
    expo start --web

O navegador abrirá com o console e com a página em execução no localhost.

===========================================================

Alternativamente, uma versão já compilada está disponível na pasta "web-build".

Com o npm instalado, basta executar o comando: 

    npx serve web-build

============================================================

Recomenda-se utilizar o navegador em "modo mobile": 
    F12 para abrir as Ferramentas de desenvolvedor, e Ctrl+Shift+M para ativar a função.
